# Wazuh SIEM Deployment

## 📌 Project Overview
This project demonstrates the installation, configuration, and basic usage of the Wazuh Security Information and Event Management (SIEM) platform on Ubuntu Linux.

The project focuses on monitoring system activities and detecting file integrity changes using Wazuh.

---

## 🎯 Objectives
- Install and configure Wazuh SIEM.
- Access the Wazuh Dashboard.
- Monitor system events.
- Detect file creation, modification, and deletion.
- Learn the basics of SIEM and File Integrity Monitoring (FIM).

---

- Ubuntu Linux
- Wazuh SIEM
- VMware Workstation
- Linux Terminal
